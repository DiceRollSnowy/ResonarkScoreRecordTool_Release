 Author: だいすろーる
 ver0.9.5 (2024/08/21)
 

【インストール方法】
任意のフォルダに配置して"ResonarkScoreRecordTool.exe"を起動してください。
Program Filesなど管理者権限が必要な場所には置かないようにしてください。

このプログラムは64bit Windowsで使うことを想定しています。
初回起動時に.NETのランタイムのインストールが必要な場合があるのでインストールしてください。


【アップデート方法について】
起動時に表示されるダイアログより、旧バージョンで使用していたデータベースファイル（ResonarkScoreRecord.sqlite3）を選択してデータ移行を押してください。
ダイアログを閉じてしまった場合は、"Tools" - "スコアデータ移行"から再度表示できます。
正常に移行された場合は、移行したデータ件数が表示されます。


【対応ワールド】 
現在はVRChatの下記ワールドに対応しています（2024/08/21時点）
- RESONARK 4 ELVES Pt.1 - RhythmGame - v4.1.2 
- RESONARK 4 ELVES Pt.2 - RhythmGame - v4.1.2 


【ファイルについて】
- ResonarkScoreRecord.sqlite3
データベースです。
楽曲データやスコアデータなどを保存しています。

- logs / ResonarkScoreRecordToolLog.txt
アプリのログファイルです。
不具合が発生した場合、ファイルを送付して頂くことがあります。

- importdb/ResonarkScoreRecord_xxx.sqlite3
スコアデータを移行した際に、選択した旧データベースファイルがコピーされるフォルダです。


【ツールの使い方】
0. データエクスポート
事前にVRChatのRESONARK4ワールドでデータエクスポートをします。
ゲーム内 "Options" - "その他" - "データエクスポート" よりjson形式のテキストをコピーできます。

1. Jsonファイルを開く
選択したファイルのスコアデータを一覧に表示します。（登録は行いません。）
クリアデータのみ表示にチェックを入れておくと、クリアした楽曲データのみ一覧に表示します。

[Point] ファイル拡張子は".txt"など".json"でなくても問題ありません。

2-1. データ登録
選択したファイルのスコアデータを登録します。
- クリアした楽曲のスコアデータのみ登録します。
- 複数のJsonファイルを同時に登録できます。
- セーブ日時、プレイヤー名、ミラーモードが同じデータは登録することができません。
- ミラーありのデータを登録したい場合は右上のラジオボタンで適切なものを選択してください。
- ミラーモードが異なる場合は、同時に登録を行わず改めて登録しなおしてください。

[Point1] ファイル拡張子は".txt"など".json"でなくても問題ありません。
[Point2] JsonファイルはUTF-8形式のみ登録できます。プレイヤー名などが文字化けする場合はテキストエディタなどで文字コードを変換してから登録してください。
[Point3] 誤って登録した場合は、スコアデータ一覧よりスコアデータを削除してください。

2-2. 登録データを表示する
登録したスコアデータを一覧に表示します。
検索・ソート条件を指定することができます。
- Data:
 * Tech-Rate: 各楽曲、プラットフォームごとに、Tech-Rateが最大のデータを表示します。
 * HighScore: 各楽曲、プラットフォームごとに、Scoreが最大のデータを表示します。
 * All: 登録したデータを全て表示します。（ClearStateを指定することはできません。）

- Difficult: 指定した難易度のデータを表示します。C.T.は疎通試験のデータです。

- Platform: 指定したプラットフォームのデータを表示します。

- ClearState: 指定したクリア状況のデータを表示します。
  * AAP            : All Arch-Perfectのデータ
  * AAP, AP        : All Arch-Perfect / All Perfectのデータ
  * AAP, AP, FC    : All Arch-Perfect / All Perfect / Full Comboのデータ
  * [Not] AAP      : クリア済みでAll Arch-Perfectでないデータ
  * [Not] AP       : クリア済みでAll Perfectでないデータ
  * [Not] FC       : クリア済みでFull Comboでないデータ
  * [Not] Clear    : クリアしてないデータ
  
- World: 指定したワールドのデータを表示します。
  
- Mirror: 
 * OFF    : ミラーオフのデータのみ表示します。
 * OFF+ON : ミラーオフのデータとオンのデータを表示します。

- Sort: プルダウンで指定した順番に並び替えしたデータを表示します。
  * ASC / DESC: "ASC"は昇順、"DESC"は降順で表示します。ボタンをクリックすると切り替えることができます。  

- 一覧の項目について
  * Title       : 楽曲名
  * Difficult   : 楽曲難易度
  * Level       : レベル
  * State       : FC（= Full Combo）、AP（= All Perfect）、AAP（= All Arch-Perfect）
  * Score       : 楽曲クリアスコア
  * Rank        : 楽曲クリアランク
  * Tech-Rate   : Tech-Rate
  * SaveDate    : データセーブ日時
  * Platform    : プラットフォーム
  * Mirror      : ミラーモード


【その他の機能】
- About
"File" - "About"より、アプリのバージョンや連絡先を確認することができます。

- 収録楽曲一覧
"Tools" - "楽曲一覧"より、収録楽曲の一覧を表示します。
楽曲名、難易度、作曲者、実装日などを確認できます。

- 登録データ一覧・削除
"Tools" - "登録データ一覧"より、過去に登録したJsonデータの一覧を表示できます。
日付、プレイヤー名、ワールド、件数などを確認できます。
また削除ボタンを押すと、登録したデータを削除することができます。（復元はできないので注意してください。）

- スコアデータ移行
"Tools" - "スコアデータ移行"よりスコアデータを移行できます。

- ResonarkChampionship4 スコア一覧
"RSC" - "ResonarkChampionship4"より、RSC4の対象楽曲のスコア一覧を表示できます。
表示ボタンを押すと、予選・本選それぞれの一覧を表示します。
Sortを選択して表示すると、MusicIDの昇順、Tech-Rateの降順、HighScoreの降順、Levelの降順でソートできます。
